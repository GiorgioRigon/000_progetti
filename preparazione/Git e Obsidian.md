# Git e Obsidian

Questo vault si trova gia dentro un repository Git:

- repository: `000_progetti`
- vault: `preparazione`

## Uso da Obsidian

1. Apri la cartella `preparazione` come vault in Obsidian.
2. Vai in `Settings` -> `Community plugins`.
3. Disattiva `Safe mode`.
4. Installa il plugin `Obsidian Git`.
5. Abilita il plugin e imposta autore, commit automatici o push/pull secondo le tue preferenze.

## Note pratiche

- I file locali di workspace di Obsidian sono ignorati da Git.
- Le note Markdown e la configurazione utile del vault restano versionate.
- Per il primo push successivo alle modifiche puoi usare sia terminale sia il plugin Obsidian Git.
