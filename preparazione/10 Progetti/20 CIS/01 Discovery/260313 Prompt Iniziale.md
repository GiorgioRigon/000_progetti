## Obiettivo

Creare un sistema di agenti per un'attività di servizio commerciale per le aziende.

## Vincoli e principi

- il controllo e l'orchestrazione restano umani
- il sistema deve automatizzare il maggior numero possibile di attività interne
- la soluzione deve essere universale e adattabile tramite moduli di personalizzazione

## Attività principali da supportare

- ricerca di potenziali clienti
- creazione di un database di nominativi e contatti
- qualificazione dei contatti
- preparazione di una strategia di contatto
- preparazione di email o messaggi a freddo
- preparazione di script per follow up
- preparazione e gestione del CRM

## Casi iniziali

- ricerca e procacciamento di concerti per un coro
- ricerca di clienti per un servizio di consulenza nella certificazione

## Stack ipotizzato

- LLM cloud come GPT o Claude per progettazione, struttura del sistema e task più pesanti
- LLM locali per scrittura email, selezione e lavoro su dati sensibili
- Python per la programmazione
- Flask per interfacce HTML locali
- SQLite per il database
- Obsidian per organizzazione di idee e memoria
- GitHub per organizzazione file e backup
