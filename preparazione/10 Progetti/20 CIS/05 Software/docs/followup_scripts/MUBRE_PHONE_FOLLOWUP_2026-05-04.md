# MU.BRE. - Script telefonata follow-up

Data target: `2026-05-04`

Obiettivo:

- verificare che la mail sia arrivata
- capire se `Loreta Busatta` e la persona giusta
- eventualmente ottenere il referente corretto
- se c'e apertura, proporre un confronto breve

## Script base

```text
Buongiorno, sono Giorgio [Cognome] di Ethics.

Avevo scritto nei giorni scorsi alla Dott.ssa Loreta Busatta in merito alla gestione del rinnovo PdR125.

Chiamavo solo per capire se la mail è arrivata correttamente e se la Dott.ssa Busatta è la persona più adatta a seguire questo tema.
```

## Versione breve

```text
Buongiorno, sono Giorgio [Cognome] di Ethics.

Avevo scritto nei giorni scorsi alla Dott.ssa Busatta sul tema PdR125.

Chiamavo solo per verificare se la mail è arrivata e se è lei la persona giusta.
```

## Se chiedono: "Di cosa si tratta?"

```text
Molto semplicemente: noi lavoriamo su parità di genere e percorsi di certificazione e, in alcuni casi, supportiamo le aziende nel rendere più semplice la gestione pratica della PdR125, soprattutto nella fase di rinnovo.
```

## Se rispondono: "Mandi una mail"

```text
Certo, la mail l’ho già inviata nei giorni scorsi. Volevo solo verificare che fosse arrivata alla persona corretta.
```

## Se rispondono: "Non è lei"

```text
Perfetto, grazie. Posso chiederle cortesemente a chi conviene inviarla o con chi è meglio che mi confronti?
```

## Se passano Loreta Busatta

```text
Buongiorno Dott.ssa Busatta, sono Giorgio [Cognome] di Ethics.

Le avevo scritto nei giorni scorsi in merito alla gestione del rinnovo PdR125. La chiamavo solo per capire se la mail è arrivata correttamente e se il tema può essere di interesse per voi, anche solo in prospettiva.
```

## Se c'è apertura

```text
Perfetto, allora non le rubo tempo adesso. Se per lei va bene, possiamo sentirci 10 minuti nei prossimi giorni e capire se il tema ha senso anche per voi.
```
